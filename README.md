# portfolio
My process is pretty simple. Firstly, I added all HTML elements to the site, remember to divide them into small groups, such as nav, banner, about, and best work, which suits my content. Secondly, I went back to the small group and worked on bootstrap to add styling. Thirdly, I moved to CSS to add media queries for responsive images and content sizes.
My challenges were getting familiar with bootstrap because I used CSS for the whole 2 semesters, which made it easy to build styling for my site. It took me a me a few days to understand Bootstrap and to know how to use their key words for searching.
What I have learned is that everything starts with the small parts; don’t rush.
